ИСПРАВЛЕНИЕ ОТЧЁТОВ ПО ДОМАШНИМ РАБОТАМ КРИСТИНЫ

Что исправлено:
1. После перезагрузки страницы снова отображаются зелёные и красные результаты проверки.
2. Домашняя работа засчитывается в прогресс после нажатия «Submit to teacher», а не только при 100% правильных ответов.
3. Кнопка отправки сначала ждёт сохранения строки в Supabase, затем отправляет Telegram-отчёт.
4. Отчёт содержит ученицу, название работы, балл, процент и количество ошибок.
5. Повторный отчёт для одной и той же отправки не дублируется.

ЧТО НУЖНО СДЕЛАТЬ

1. На GitHub заменить:
   - app.js
   - config.js
   - supabase/functions/notify-telegram/index.ts

2. В Supabase Dashboard -> SQL Editor выполнить файл:
   supabase/homework-reports.sql

3. После загрузки файлов на GitHub открыть GitHub -> Actions и запустить:
   «1 - Setup Telegram notifications»
   Это повторно развернёт обновлённую функцию notify-telegram.

4. Проверить отправку:
   - открыть домашнюю работу;
   - нажать Check answers;
   - нажать Submit to teacher;
   - дождаться сообщения об успешной отправке отчёта.

Важно: уведомления о новых материалах продолжат работать через ту же Edge Function.
